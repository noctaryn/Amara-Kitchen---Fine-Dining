# Amara Kitchen

A modern and responsive fine dining restaurant website concept designed to showcase an upscale culinary experience. The project features a sophisticated user interface, interactive navigation, menu presentation, and reservation section.

## Features
- Responsive design for desktop and mobile devices
- Elegant fine dining themed UI/UX
- Interactive menu showcase
- Reservation section
- Modern typography and visual styling

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Live Demo
https://tubular-profiterole-b4d8ba.netlify.app/

## Purpose
This project was created to practice frontend web development, responsive design, and modern user interface design principles.
